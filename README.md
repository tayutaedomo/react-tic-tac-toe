## Set up

```
$ git clone git@github.com:tayutaedomo/react-tic-tac-toe.git
$ cd react-tic-tac-toe
$ touch .bash_history
$ docker-compose build
$ docker-compose run --rm app yarn install
$ docker-compose up
```
